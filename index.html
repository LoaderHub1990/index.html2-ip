วิธีการใช้งาน

1. ใช้คำสั่ง /voiceonline เพื่อส่งเมนูไปยังช่องที่ต้องการ

2. ในเมนูให้คลิก "ตั้งค่าข้อมูล" เพื่อกรอกโทเค่น, Server ID, Voice Channel ID

3. เมื่อตั้งค่าเสร็จ คลิก "จัดการออนช่องเสียง" เพื่อเริ่ม/หยุดการเชื่อมต่อ

4. ระบบมี Auto-Restore เมื่อบอทเริ่มใหม่จะเชื่อมต่ออัตโนมัติ

⚠️ โทเค่นจะถูกเก็บไว้ในเครื่องคุณเท่านั้น
